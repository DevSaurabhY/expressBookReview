# Express Book Review API

This project is part of the IBM Developer Skills Network course.

## 📚 Project Description
This is a Node.js and Express-based REST API that allows users to:

- Get the list of all books
- Search books by ISBN
- Search books by Author
- Search books by Title
- Register new users
- Login registered users
- Add a review to a book
- Modify a book review
- Delete a book review
- Get book reviews

The project uses a JSON file as the data source.

---

## 🚀 Technologies Used

- Node.js
- Express.js
- JSON
- REST API

---

## 🔧 How to Run the Project

1. Clone the repository
2. Run `npm install`
3. Run `node index.js`
4. Open browser or Postman and test the endpoints
